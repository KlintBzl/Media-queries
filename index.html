Usando Media Queries

O que é?

Media Queries é uma técnica que usamos para atribuir diferentes estilos de CSS para diferentes tamanhos de telas, assim, fazendo com que mude o jeito a cada tela detectada. Então, toda vez que você mexer em seu CSS em uma resolução diferente, seja, tela do celular ou computador, ela vai mudar de acordo com as modificações específicas.

Como usar:

O media queries utiliza o comando @media para mudar o estilo de uma página web, colocando uma condição para caso o dispositivo que esteja acessando a página for diferente “de celular para computador” a página vai mudar e ficar mais formal ao olhar das outras pessoas.
